## 問題
### for型態為無號整數
這裡的i因為是無號整數，所以扣到0在扣一次就溢位了

```cpp
    for(unsigned i = 2; i >= 0; --i) {
        cout << "i=" << i << endl;
    }
```

解決辦法，使用

```cpp
    for(unsigned i = 3; i > 0; --i) {
        cout << "i=" << i-1 << endl;
    }
```

### 副程式預設數值
####實作不拆開
- 直接在寫在頭文件宣告
- 定義可直接省略參數

####實作拆開
- 不可寫在頭文件上
- 基於第一項，拆分的定義不可省略參數
- 只能在main上省略參數
